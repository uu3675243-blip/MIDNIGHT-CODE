📖 ゲーム概要と遊び方

🎮 どんなゲーム？
あなたは「怪盗」となり、仲間と一緒に金庫を解錠するミッションに挑みます。
各金庫には「暗証番号（目標値）」が設定されており、プレイヤーは手札から数字カードを伏せて配置します。規定枚数に達すると金庫が開示され、合計値が目標と一致すれば💎解錠成功！超えたら💥爆発！ 足りなければ❌不足で失敗。
相手の思考を読み、ブラフを仕掛け、時にはなすりつけ合いながら、最多チップを目指します。

🕹️ 操作方法

  1. ロビーで「🆕部屋を作る」か「🚪入室」（ROOM ID入力）
  2. ゲーム画面で手札から1枚タップして選択
  3. 次に配置したい金庫をタップして伏せる（「✅配置ボタン」でも確定可）
  4. 🎯予告状ONで配置 = 成功時に報酬独占/失敗時-5chip（1金庫1人まで先着）
  5. 🔭スカウトボタン→ 場の伏せカードをタップで2.5秒覗き見（1ゲーム1回、ジャスト成功で復活）
  6. 🏃パスで手札を出さずに終了（手札0枚時など）

🎉 3〜4人 パーティモード: 混沌の中で責任のなすりつけ合い！ バースト累積3回で「🏆伝説のドジ怪盗」発動権（全員のチップを平均化する逆転の一手）

🎯 2人 デュエルモード: 初期👻ゴーストカード+毎ターン🎲ノイズ配置で不確定要素を演出。相手に5点以上差をつけられたら💱コード・スワップで丸ごと入れ替え可能

🎰 ラスト・ルーレット（パーティ限定）: 山札残り5枚以下で発動候補。初回100%、以降30%の確率で適用され、対象金庫の報酬・ペナルティが×2倍

⚡ サドンデス: 山札枯渇と同時に最終局面へ。ゴースト供給と新規金庫補充が停止される

👁️ 観戦モード: 5人目以降 / 満員 / ゲーム進行中の入室は自動で観戦者になります（上限20名）

📖 MIDNIGHT CODE 完全ルール

🎮 ゲームの目的

ブラフ＆カウンティング心理戦ボードゲーム。怪盗となって仲間と金庫を解錠し、最多チップを目指します。手札を伏せて配置するため、誰が何を置いたかが隠される点が要。

🕹️ 操作フロー

  1. 手札から1枚タップ（選択）
  2. 配置先の金庫をタップ（または「✅配置」ボタン）
  3. 必要なら🎯予告状ON、🔭スカウト、🏃パスを併用
  4. 金庫規定枚数到達 → 自動開示・判定
  5. 判定結果に応じチップ増減、金庫補充、次ターンへ

🎯 勝利条件

ゲーム終了時、所持チップが最多のプレイヤーが勝利。

🧮 判定結果

結果条件小中大
💎成功合計=目標+3均等+5均等+10均等
🎯ニアピン合計=目標-1+1+2+5
💥バースト目標超犯人-2犯人-2犯人-3
❌不足≤目標-2全員報酬なし、金庫破棄

バーストの犯人定義: 最後にカードを置いた人（判定の引き金）が犯人。犯人以外は自分が出した数値の合計分だけチップ獲得（おこぼれ）。

ジャスト特典: 💎成功時、最後に置いた人の🔭スカウトチップが復活（未使用なら影響なし）。

🎯 予告状

配置時にONで宣言。1つの金庫に1人まで（先着順、後から同じ金庫には宣言不可）。
  • 💎/🎯: 報酬を宣言者が独占（他プレイヤーも参加はできる）
  • 💥: 通常ペナルティに加え宣言者は-5

🎉 パーティモード（3-4名）

伝説のドジ怪盗: 累積バースト3回で発動権取得、任意で全員のチップを平均値にできる。

🎯 デュエルモード（2名専用）

  • 各金庫前に山札から1枚ゴーストを伏せて配置（合計枚数は同じ：小3/中4/大5）
  • 手番 = 手札から1枚配置 + 山札から1枚引いてノイズ配置（見ずに好きな金庫へ）+ 手札補充
  • ゴースト/ノイズカードでのバースト → 置いたプレイヤーが犯人
  • 🔭スカウト: 相手のカード or ゴースト/ノイズ を覗ける
  • 🎯予告状: 相手が先に宣言した金庫には宣言不可
  • 💰ノイズ配置パス（累進課税）: 1枚目=1chip / 2枚目以降=2chipを支払いでパス
  • 💱コード・スワップ（最終スコア）: 相手に5点以上差をつけられた場合、最終スコアを丸ごと入れ替え可能
  • 💱コード・スワップ（場のカード）: 同一手番でバースト2回以上を起こした次ターン、進行中の金庫にあるカード2枚を入れ替え可能

💥 特殊イベント

  • 2金庫同時爆発: 同一手番内で2つ以上の金庫を同時爆発させた場合、犯人にペナルティ相殺ボーナス（+3chip）と「コード・スワップ」発動権を付与
  • 💱 コード・スワップ（場のカード交換）: 現在進行中の金庫にある任意のカード2枚（他人のカード・裏向き含む）を入れ替え。手札との交換は不可。
  • 奇跡の解錠（全ゴースト成功）: 金庫内カードが全てゴースト/ノイズで解錠成功 → 最後に置いたプレイヤーに +3chip限定ボーナス（手札調整権は付与しない）
  • 🎴 手札調整権（パーティ限定）: 解錠成功時、最後に置いた人に1枚ドロー&1ディスカードの権利を付与
  • 🎰 ラスト・ルーレット（パーティ限定）: 山札残り5枚以下で金庫補充時に適用（初回100%/以降30%）。適用金庫の報酬・バーストペナルティが×2
  • ⚡ サドンデス: 山札枯渇でゴースト供給と新規金庫補充が停止。現在場の金庫を処理し切った時点でゲーム終了

🏁 終了条件

金庫山が尽きかつ場の金庫も全て解決 → ゲーム終了。またはサドンデス進行中に全金庫を処理し切った時点。
